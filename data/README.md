# data/

This folder holds input data for the pipeline. Only **public reference data**
ships here; the challenge data is not redistributed (see ../DATA.md).

## Shipped

```
external/
├── sjr.csv         SCImago Journal Rank metrics        (used by 02_external_features.R)
├── topfactor.csv   TOP Factor journal transparency     (used by 02_external_features.R)
└── flora.csv       FLoRA replication records    (author-overlap source)
```

## You must add (to run Stage 1 — see ../DATA.md)

```
Round1/  Round2/  Round3/        challenge round datasets (claim text, metadata, labels)
Papers Trainingdata/             source manuscripts (PDF)
tei_xml/                         GROBID full-text XML (from the _ft companion repo)
FORRT_Training_Data.csv          FORRT training table
dataset_incl_round2.csv          assembled R1+R2 claim/label table
```

## Generated by the pipeline (not shipped; rebuilt locally)

```
train_raw.rds, test_raw.rds      ← 01_data_raw_assembly.R
external_features.rds, cache/    ← 02_external_features.R
cache/llm_features.rds           ← 03_llm_features.R
train_base.rds, test_base.rds    ← 04_base_dataset.R   (the model input matrix)
```
