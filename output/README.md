# output/

Shareable result artifacts (our model outputs; no challenge data).

| File | What |
|---|---|
| `submission.csv` | the deliverable: `claim_id, m1, m2, m3` for the 45 R3 claims |
| `r3_elnet_plus_predictor.csv` | deployed **m1** (feature model) per claim — from `09_final_comparison.R` |
| `r3_prior_shrinkage_predictor.csv` | deployed **m2** (prior × evidence): `prior_3fam, p_base, pred` |
| `submission_diagnostics.csv` | trio cross-correlations + per-slot distribution summary |

`submission.csv` and `submission_diagnostics.csv` are regenerated by
`Rscript pipeline/10_submission.R` (see ../REPRODUCE.md, Level 1).
